# Proyecto Mazinger Z 🤖🚀

Este proyecto es una aplicación web interactiva dedicada a la legendaria serie **Mazinger Z**. Permite explorar episodios, bestias mecánicas, personajes y jugar con una baraja de cartas personalizada.

## Características

- 🃏 **Juego de Cartas**: Colecciona e interactúa con cartas de personajes y bestias.
- 📺 **Gestor de Episodios**: Organización y visualización de la serie (requiere archivos locales).
- 🎵 **Banda Sonora**: Reproductor con ecualizador dinámico.
- 📱 **PWA Ready**: Instalable como aplicación web en móviles y escritorio.
- 🔍 **SEO Optimizado**: Metadatos completos y etiquetas OG para mejor visibilidad en Google y redes sociales.
- 🌐 **Redes Sociales**: Integración de enlaces sociales unificados en el pie de página.
- 🛠️ **Scripts de Procesamiento**: Herramientas en PowerShell para conversión y reescalado de video a 4K usando FFmpeg.

## Requisitos de Instalación

1.  Clona el repositorio.
2.  Para usar los scripts de video, asegúrate de tener `ffmpeg` en la carpeta `bin/` o instalado globalmente en el sistema.
3.  Abre `index.html` en un navegador moderno.

## Nota sobre Media

> [!IMPORTANT]
> Por razones de derechos de autor, los episodios de video y ciertos archivos de audio **no están incluidos** en este repositorio. Los scripts proporcionados ayudan a optimizar tus propios archivos legales.
